## Edibleとは
『Edible』は規格外農産物を減らし、持続可能な社会の実現を目指します。 

味が同じでも形が悪いため、棄てられる農産物は全体の2-3割存在します。   
本サービスはそれら規格外農産物を減らし、世の中のフードロスを減らします。   
さらにSDGs Goal12の『つくる責任、つかう責任』達成に則しています。
